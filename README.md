# Expense Tracker

This is a simple expense tracker I built in Python to practice programming concepts such as file handling, functions, and working with CSV files.

## Features

- Add new expenses
- View all recorded expenses
- Calculate total spending
- Display spending by category
- Store data using CSV files

## Technologies Used

- Python
- CSV module

## How to Run

1. Download the project files.
2. Open a terminal in the project folder.
3. Run:

```bash
python expense_tracker.py
```

## Why I Built This

I created this project to apply the Python concepts I learned and gain experience building a complete program from scratch.
